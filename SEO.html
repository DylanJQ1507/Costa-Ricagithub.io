Posiciona TU web con SEO

¿Que es SEO?

SEO significa Search Engine Optimization (Optimización para motores de búsqueda).
Se trata del conjunto de estrategias y técnicas de optimización que se hacen en una página web para que aparezca orgánicamente en buscadores de Internet como Google, Yahoo o Youtube
Con el SEO u Optimización para Motores de Búsqueda — del inglés Search Engine Optimization— puedes hacer que tu sitio web aparezca en búsquedas relevantes, conectando tus contenidos con las personas que están buscando productos o servicios como el que ofreces.

Como optimizar tu web para SEO

para optimizarla debemos saber que google utiliza "arañas" web para buscar entre las miles de paginas web para mostrarlas segun la busqueda del usuario
para que estos robots posicionen nuestra web debemos utilizar el SEO
para esto debemos cumplir requisitos como la buena utilizacion de etiquetas, la utilizacion de un buen nivel semantico y descripcion de nuestra pagina

Tu pagina debe cumplir con alguno de estos requisitos
  1. Maquetacion sin tablas
  2. Un codigo corto y funcional
  3. Optimizar Texto
  4. Responsiva

META Tags

estas etiquetas dan informacion de nuestra web son solo para el codigo el usuario no lo ve en la pagina, su relacion es ayudar al SEO a posicionar nuestra pagina en el motor de busqueda apartir de palabras clave y descripciones

Tipos de meta tags

charset

le dice a el documento la codificacion a utilizar

Meta description

el contenido o tema del que trata nuestra pagina

viewPort

adapta nuestra pagina al dispostivo del cual se ingresa

Meta keywords

tendra las palabras claves de nuestra pagina web

Meta autor

describe al autor de la pagina

Meta robots

para que los motores de busqueda sepan que hacer con nuestra pagina

Meta OG

permite que cuando se pegue el enlace de nuesta pagina en una red social mostrar cosas de la pagina, como imagen o descripcion
